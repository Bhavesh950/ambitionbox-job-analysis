# 🚀 AmbitionBox Job Analysis

A Flask-based web application that allows users to explore, filter, and analyze companies based on salary, ratings, location, and industry using real-world data.

---

## 🌟 Features

* 🔍 Filter companies by:

  * Location
  * Expected Salary
  * Industry
  * Minimum Rating

* 📊 Output Options:

  * Table View
  * Interactive Visualizations

* 🌙 Dark Mode Support

* 📥 Download filtered data as CSV

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS, Bootstrap
* **Data Processing:** Pandas, NumPy
* **Visualization:** Plotly

## ⚙️ Installation & Setup

```bash
git clone https://github.com/Bhavesh950/ambitionbox-job-analysis.git
cd ambitionbox-job-analysis
pip install -r requirements.txt
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```
